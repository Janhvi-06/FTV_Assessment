# FTV Landing Page Assessment

This project is the landing page created as part of the assessment provided after my interview.  
The task involved building a  landing page using HTML and CSS, along with some basic animations and visual effects.

## Features
- Custom animations & hover effects  
- Clean UI with organized structure  
- Cross-browser compatible

## Tech Stack
- HTML5  
- CSS3  

## How to Run
1. Download or clone the repository  
2. Open `index.html` in any browser  
3. All assets (images, CSS, external fonts) are included in the project folder
